jQuery.ajax

Make an HTTP request asynchronously without refreshing the page (e.g. a postback)

jQuery.ajax( settings );

Version 1.0

Argument Description
settings An object that contains key/value pairs of configuration data for this Ajax request. All settings are optional. A default can be set up globally for any option by using $.ajaxSetup(). 
jQuery.ajax( url, /* optional: */ settings );

Version 1.5
