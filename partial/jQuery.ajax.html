jQuery.ajax

Make an HTTP request asynchronously without refreshing the page (e.g. a postback)

jQuery.ajax( settings );

Version 1.0

settings
An object that contains key/value pairs of configuration data for this Ajax request. All settings are optional. You can set and change the global default for any option by using $.ajaxSetup().
Setting Description
accepts Default: depends on dataType
Accepts: Object Modify the mapping of content types to dataTypes. javascript $.ajax({ accepts: { json: "text/javascript, application/json", xml: "text/xml" } }); 
jQuery.ajax( url, /* optional: */ settings );
